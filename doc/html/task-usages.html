Apkrovų skaičiavimas

  1. Intervalo nustatymo įrankių juostoje pasirinkti norimas intervalo pradžios datą.
  2. Intervalo nustatymo įrankių juostoje pasirinkti norimas intervalo pabaigos datą.
  3. Intervalo nustatymo įrankių juostoje pasirinkti į kokius mažesnius intervalus skaidyti pasirinktą intervalą.
  4. Spausti mygtuką „Rodyti“.

Prognozių skaičiavimas

  1. Pasirinkti kortelę „Planuojami kiekiai“.
  2. Kortelėje „Planuojami kiekiai“ suvesti planuojamus paramos priemonių kiekius.
  3. Intervalo nustatymo įrankių juostoje pasirinkti norimas intervalo pradžios datą.
  4. Intervalo nustatymo įrankių juostoje pasirinkti norimas intervalo pabaigos datą.
  5. Intervalo nustatymo įrankių juostoje pasirinkti į kokius mažesnius intervalus skaidyti pasirinktą intervalą.
  6. Spausti mygtuką „Rodyti“.
