Šioje sąsajoje galima:

  • Sukurti naują naudotojo paskyrą.
  • Pašalinti egzistuojančio naudotojo paskyrą.
  • Pakeisti naudotojo privilegijas arba slaptažodį.

Dėl PostgreSQL DBVS apribojimų, keisti sukurto naudotojo vardą galimybės nėra. Norint tai padaryti reikia pašalinti esamą naudotoją ir sukurti naują su norimu vardu
